## **1. Introduction**
Albukhr Investment Limited ("Albukhr", "we", "our") is committed to protecting the personal information of our users within the Pi Network ecosystem. This policy explains how we collect, use, store, protect, and share your information.

We follow Pi Network guidelines and prioritize user trust, transparency, and security.

---

## **2. Data We Collect**
We only collect **Pi SDK data** and information voluntarily provided by the user.

### **2.1. Data from Pi Login**
- Pi User ID  
- Pi Username  
- Country or language preference  
- Profile picture (optional)  
- Wallet address (only when required)

### **2.2. Data Provided by the User**
- Selected projects or investments  
- Staking duration  
- Preferences  
- Feedback or messages  

### **2.3. Data We DO NOT Collect**
- Phone numbers  
- Emails  
- Passwords  
- GPS or location data  
- Device identifiers  
- Contacts  

---

## **3. How We Use Your Data**
We use data **only** for:

- Identity verification via Pi Login  
- Managing staking or project participation  
- Displaying your dashboard  
- Calculating contributions or rewards  
- Support and communication  
- Improving app performance  

We **never sell, rent, or trade** personal data.

---

## **4. Data Storage**
Your data is stored securely using:

- Encrypted servers  
- HTTPS communication  
- Access restrictions  
- Regular backups  

Only authorized personnel can access data when necessary.

---

## **5. Data Sharing**
We only share data with:

- Pi Network systems (for verification/blockchain actions)  
- Albukhr internal systems  

We do **not** share data with:

- Advertisers  
- External marketing services  
- Data brokers  

---

## **6. User Rights**
Users have the right to:

- Request data deletion  
- Request data correction  
- View the data collected  
- Withdraw consent  

If you revoke Pi Login permissions, Albukhr will delete all associated personal data.

---

## **7. Data Retention**
We retain data only for:

- Ongoing use inside the app  
- Active staking calculations  
- Pi Network compliance needs  

If a user deletes their account or access is revoked, all data is safely removed.

---

## **8. Security Measures**
Our protection measures include:

- Encryption in transit and at rest  
- Role-based access control  
- Anti-fraud checks  
- Secure API communication  
- Blockchain-level security

We may update this policy occasionally. Changes will be announced inside the app.
